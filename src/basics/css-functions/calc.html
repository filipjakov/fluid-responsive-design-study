calc()

The calc() CSS function lets you perform calculations when specifying CSS property values.

calc() is for values

The only place you can use the calc() function is in values:

  • Can be used in property values: 
    .el {
  width: calc(100% - 20px);
  height: calc(100vh - 20px);
}
  • Can be used as a part of a property: 
    .el {
  margin: 10px calc(2vw + 5px);
  border-radius: 15px calc(15px / 3) 4px 2px;
  transition: transform calc(1s - 120ms);
}
  • can even be a part of another function that forms a part of a property 
    .el {
  background: #1E88E5 linear-gradient(
    to bottom,
    #1E88E5,
    #1E88E5 calc(50% - 10px),
    #3949AB calc(50% + 10px),
    #3949AB
  );
}

calc() is for lengths and other numeric things

  • A nope: 
    .el {
  /* Nope! */
  counter-reset: calc("My " + "counter");
}
  • Can work with unitless numbers: 
    .el {
  line-height: calc(1.2 * 1.2);
}
  • You can also not perform any calculation and it is still valid: 
    .el {
  /* Weird flex but OK */
  width: calc(20px);
}

Mixing units

.el {
  width: calc(100% - 20px);
  height: calc(100vh - 20vmin);
  transform: rotate(calc(1turn + 45deg));
  animation-delay: calc(1s + 15ms);
}

Some rules:

NOTE: Invalid values invalidate the whole individual declaration

  • Addition (+) and subtraction (-) require both numbers to be lengths,
  • Division (/) requires the second number to be unitless,
  • Multiplication (*) requires one of the numbers to be unitless,
  • Whitespace matters: 
    .el {
  /* Valid üëç */
  font-size: calc(3vw + 2px);

  /* Invalid üëé */
  font-size: calc(3vw+2px);

  /* Valid üëç */
  font-size: calc(3vw - 2px);

  /* Invalid üëé */
  font-size: calc(3vw-2px);
}
  • You can nest calc()s but it's usually not necessary: 
    .el {
  /* üëç */
  width: calc((100% / 3) - (1rem * 2));
}
